Principais Formatações

Negrito ou destaque

Nessa frase temos um termo em negrito usando a tag < b > (não semântica).

Nessa frase temos um termo em destaque usando a trag < strong > (semântica).

Itálico ou Ênfase

Nessa frase temos um termo em itálico usando a tag < i > (não semântica).

Nessa frase temos um termo enfático usando a tag < em > (semântica).

Texto Marcado

Podemos marcar um texto utilizando a tag < mark >.

Texto Pequeno

Podemos colocar um texto com letras pequenas utilizando a tag < small >.

Texto Deletado

Podemos riscar um texto desconsiderado utilizando a tag < del >.

Texto Inserido

Podemos sublinhar um texto inserido utilizando a tag < ins >.

Texto Sobrescrito

Para inserir coisas do tipo x25 + 30 utilizamos a tag < sup >.

Texto Subscrito

Para inserir coisas do tipo H2O utilizamos a tag < sub >.
