Usando Links

Externos

Para acessar outros sites através de um link, utilizamos a tag <a href= "link-externo.com" target="_blank" rel="external">
Ex:
Você pode acessar o youtube, o instagram e qualquer outro site.

Internos

Para acessar links internos, como arquivos html, utilizamos a tag < a href="caminho ou nome do arquivo" rel="next ou prev" >
Ex:
Área de Vídeos
Área de notícias
Este site é patrocinado pelo © Google

Downloads

Para criar links de download é necessário a utilização da tag < a href="#" download="nome-do-arquivo" type="depende do formato do arquivo (site de apoio: iana.org/assignments/media-types/) >
Ex:

  • Download .PDF
  • Download .RAR
