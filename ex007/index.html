Outras Formatações

Código-fonte

Para digitarmos um código e termos uma melhor visualização dele, utilizamos a tag < code >.
Ex: document.getElementById('teste')

Pré-formatação

Para escrever linhas completas de código, além de utilizarmos a tag < code >, utilizamos a tag < pre >.
Ex: Programando em python 

                
num = int(input('Digite um número: '))
    if num % 2 == 0:
print('O número digitado é Par!')
    else:
print('O número digitado é Ímpar!')
    print('Fim do programa!')

Citação Simples

Utilizamos a tag < q > para citações menores.
Ex: Como diria o pai de um amigo : O computador é um burro muito rápido.

Citações Completas

Utilizamos a tag < blockquote > para citações maiores ou completas.
Ex:
Segundo Werther Samarão, autor do livro "A necessidade faz o homem" :

O homem sempre será moldado por aquilo que ele necessita, seja em sua vida profissional, acadêmica ou até mesmo de sobrevivência. Apenas os fortes são capazes de continuar sua jornada a fim de uma melhor condição de vida e principalmente por aquilo que o faz estar vivo, a reprodução de sua espécie.

Abreviações

Para apresentarmos os significados de siglas e abreviações, utilizamos a tag < abbr >.
Ex: Estou aprendendo HTML e CSS.
