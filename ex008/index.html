Trabalhando com Listas

Listas Ordenadas

Para listarmos itens de forma ordenada, utilizamos a tag < ol > para criar o bloco de lista e a tag < li > para listá-los e ordená-los.
Ex:

  1. Acordar
  2. Escovar os dentes
  3. Tomar Café
  4. Se arrumar
  5. Ir para a escola

Listas Não-Ordenadas

Para listarmos itens de uma forma não ordenada, utilizamos a tag < ul > para criar o bloco de lista e a tag < li > para listá-los.
Ex:

  • Arroz
  • Feijão
  • Tomate
  • Cebola
  • Tempero

Listas Mistas ou Sub-Listas

Ex: Criar listas dentro de listas com as tags < ol >/< ul > e < li >.

  1. Jogos de FPS
    • Counter Strike GO
    • Call of Duty
    • Battlefield
  2. Jogos de Esporte
    • FIFA
    • NBA 2K
    • Formula 1
  3. Jogos de Luta
    • UFC Undisputed
    • Tekken
    • Street Fighter

Listas de Definição

Para listar itens com suas definições, como em um dicionário, é utilizado a tag < dl > para criar o bloco lista de definição, a tag < dt > para definir o termo que será descrito e a tag < dd > para a sua descrição.
Ex:

HTML
É uma linguagem de marcação a fim de personalizar e adicionar conteúdo ao site.
CSS
É uma linguagem de marcação a fim de personalizar e adicionar um design ao site.
JavaScript
É uma linguagem de programação a fim de personalizar e adicionar interatividade ao site.
